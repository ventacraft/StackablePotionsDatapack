# **_Stackable Potions Datapack_**
## By VentaCraft

This Datapack makes potions being stackable up to 16, but splash and lingering potions have a 3 second cooldown. Even potion that were made before are going to stack.
The Datapack is based on the mod "_StackablePotions_" that is not updated nowadays.

# **_1.0_**

The version 1.0 is for Minecraft 1.21.10, the potions stack to 16, with the 3 second cooldown, but if a potion is on a chest, it isn't going to update.

# **_1.1_**

The version 1.1 is for Minecraft 1.21.10, now, when you look to a chest (max 5 blocks), the potions also update
